# Playlab-Backend
This is a simple parser written in ruby using regex based pattern matching.

## Dependencies
        * Ruby Version  : 2.1.2
        * rspec (for testing)


## Execution
        $ ruby -r "./parser.rb" -e "Parser.new 'sample.log'"  # replace sample.log with the desired file's path
    
## Testing
        $ rspec spec/parser_spec.rb
